# waterRPA-FUKUA
对b站up不高兴就喝水的开源软件waterRPA进行了一些修改，增加了一些我认为用得上的功能
================================================================
不高兴就喝水 RPA配置工具 - 更新日志
By:b站浮夸HG

**1. 从“纯手动”到“全息录制”**
* **旧版**：所有指令、坐标、时间只能靠人脑算、手动打字输入。
* **新版**：引入了底层全局 Hook 录制。只需点一下，自动精准记录所有鼠标点击、拖拽、滚轮、键盘全按键操作，并自动换算好每一步的等待时间生成脚本。

**2. 视觉引擎“降维打击”**
* **旧版**：使用最基础的 PyAutoGUI 原生找图，慢、卡、不支持缩放、找不到就死等。
* **新版**：重构为 OpenCV 内存预加载引擎。找图极快，新增了**指定区域搜索**（解放CPU）、**多倍率缩放匹配**（无视图片忽大忽小）、以及**自定义检测频率**。

**3. 多开与本地配置方案管理**
* **旧版**：只有一个默认列表，想换个任务只能全部删掉重写，或者手动一个个导入导出文件。
* **新版**：内嵌了一套完整的“配置方案库”。你可以建几十个不同名字的任务（自带重命名、排序），一键无缝切换所有指令和参数；配置储存在本地 `config.ini`，变成了真正的绿色免安装软件。

**4. 绝对安全的“看门狗”与防封机制**
* **旧版**：一旦鼠标被脚本劫持无限死循环，几乎无法停止，只能强行关机。
* **新版**：拥有独立的“看门狗”线程。随时可以按 `ESC`、`鼠标中键` 或甩鼠标到右上角强行急停。此外还加入了点击后“主动避让”的防遮挡机制和“拟人按住时长”。

**5. 灵活的循环与倍速体系**
* **旧版**：只能选“执行一次”或“无限循环”。
* **新版**：可以精准设定循环 N 次，或者挂机“指定小时/分钟/秒”后自动停止；自带**倍速执行**，嫌录制的动作慢了直接一键加速。

**6. 现代化交互与防呆机制**
* **新版**独占的优化：
    * 指令支持**鼠标直接拖拽上下排序**（旧版不能调顺序）。
    * 加入了全局的**启动/停止热键**（F9/F10）。
    * 智能分屏日志栏，支持三级过滤（简易/详细/完全）。
    * **语法预检**：启动瞬间自动查错，填错坐标直接报错拦截，杜绝运行中途崩溃。
